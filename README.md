# 🎬 Movie Recommendation System

A content-based movie recommendation system built with Python, Scikit-learn, and Streamlit.

## Features
- Recommends 5 similar movies based on genre
- Simple and clean web UI using Streamlit
- Built using MovieLens dataset

## Tech Stack
- Python
- Pandas
- Scikit-learn (Cosine Similarity)
- Streamlit

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Dataset
MovieLens Small Dataset - 9000 movies, 100,000 ratings
